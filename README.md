# My personal website

## Description

This is my personal website, featuring articles about various tech projects I've created and a blog with similar subject matter.

## Functionality

GitHub Pages only allows for static content. Rather than copy and paste code or use a site generator like Jekyll, I used vanilla JavaScript with AJAX calls to generate certain content after page loading so that the page source could be modular (separate footer, header, banner, etc.). This happens in `main.js`.
